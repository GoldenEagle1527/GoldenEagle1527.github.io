\### 1. 引入样式

在 `<head>` 标签内添加：

```html

<link rel="stylesheet" href="../modules/qr-popup/css/qr-popup.css">

```



\### 2. 引入脚本

在 `<body>` 标签结束前添加（注意顺序）：

```html

<!-- 先引入依赖库 -->

<script src="../modules/qr-popup/js/qrcode.min.js"></script>

<!-- 再引入功能模块 -->

<script src="../modules/qr-popup/js/qr-popup.js"></script>

```



\### 3. 调用弹窗

在 JS 事件中执行：

```javascript

// 传入任意 URL 字符串

window.QrPopupModule.show("https://你的链接...");



// 或传入当前页面 URL

window.QrPopupModule.show(window.location.href);

```
